# Задача [0560]: [subarray-sum-equals-k]

**Дата:** 2026-03-03
**Сложность:** Medium
**Паттерн:** префиксные суммы, подсчет частоты, хеш-таблица

## Концепция
Given an array of integers nums and an integer k, return the total number of subarrays whose sum equals to k.
A subarray is a contiguous non-empty sequence of elements within an array.

если разность сумм (j превосходит i) от 0 до i-того элемента и от 0 до j-того равна k, это значит, что сумма от i-того до j-того равна k. оптимальный способ решения подразумевает под собой хранение сумм как ключей для каждого элемента в массиве, а в качестве значения выступает их количество (если для i,j,k-того элементов сумма равна s, тогда в хеш-таблице будет следующее: hm = {s: 3}). стоит помнить, что в хеш-таблице должна быть сумма 0, которая встречается нам в самом начале, поэтому объявляем первый элемент хеш-таблицы: {0: 1} во время ее создания.

## Ключевая идея
хранение суммы для каждого элемента массива, и сколько раз эта сумма встречается. это позволяет решить задачу за O(n) по времени.

## Пошаговый разбор
1. создаем хеш-таблицу с учетом нулевой суммы, переменную для отслеживания текущей суммы, переменную-счетчик для результата.
2. проходимся по списку, сначала считаем текущую сумму s.current, затем проверяем, встречали ли раньше такою сумму s, что s.current-k==s, и если условие срабатывает, то в результирующую переменную добавляем значение по ключу s (количество таких s в списке сумм для каждого элемента).
3. добавляем в таблицу текущую сумму с помощью метода .get, чтобы избежать ошибок. если сумма уже встречалась, тогда количество таких сумм увеличиваем на 1. после цикла возвращаем результат

## Моё решение
```python
class Solution(object):
    def subarraySum(self, nums, k):
        """
        :type nums: List[int]
        :type k: int
        :rtype: int
        """
        seen = {0:1}
        summ=0
        res=0
        for i in nums:
            summ += i
            current = summ - k
            if current in seen:
                res+=seen[current]
            seen[summ] = seen.get(summ, 0) +1
        return res

# Задача [125]: [valid-palindrome]

**Дата:** 2026-03-06
**Сложность:** Easy
**Паттерн:** two-pointers

## Концепция
пройтись по строке с двух сторон, игнорируя пробелы и знаки препинания, попутно сравнивая символы. если символы не совпали, значит строка не является палиндромом, если цикл прошел полностью, значит строка - палиндром.

## Ключевая идея
указатели будут сходиться и для решения достаточно условия строго превосходства одного над другим, например, если строка имеет четное количество элементов, тогда указатели сойдутся на двух соседних, а если нечетное тогда между ними окажется один элемент, который бессмысленно проверять - у него нет пары.

## Пошаговый разбор
1. ставим указатели на свои места.
2. используем while со строгим условием сравнения.
3. проверяем на равность.

## Моё решение
```python
class Solution(object):
    def isPalindrome(self, s):
        """
        :type s: str
        :rtype: bool
        """
        if not s:
            return True
        left = 0
        right = len(s)-1
        while left < right:
            while left < right and not s[left].isalnum():
                left+=1
            while left < right and not s[right].isalnum():
                right-=1
            if s[left].lower()!=s[right].lower():
                return False
            left+=1
            right-=1
        return True

# Префиксные суммы (Prefix Sum)

## Когда применять
- Когда нужно быстро вычислять сумму элементов в любом подмассиве (за O(1))
- Когда ищем количество подмассивов с суммой, равной заданному числу k
- Когда работаем с накопительными итогами (cumulative sums)
- Когда нужно отвечать на множество запросов суммы на отрезке
- Когда массив не меняется между запросами (статичный)

## Основная идея
Заранее вычисляем массив префиксных сумм, где prefix[i] = сумма элементов от 0 до i. Тогда сумма любого подмассива от l до r вычисляется за O(1) как разность двух префиксных сумм.

Для задач типа "найти количество подмассивов с суммой k" мы комбинируем префиксные суммы с хеш-таблицей: по мере прохода по массиву храним все встреченные суммы и их количество, и для каждого нового элемента проверяем, встречалась ли ранее сумма (текущая - k).

## Ключевая формула/инвариант
prefix[i] = prefix[i-1] + nums[i] # построение
sum(l..r) = prefix[r] - prefix[l-1] (если l > 0) # запрос суммы

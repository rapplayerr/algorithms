# [Подсчет частоты]

## Когда применять
подсчет дубликатов, отслеживание количества вхождений, проверка анаграмм, поиск уникального элемента...

## Основная идея
хранить элемент как ключ, а значением сделать его частоту в массиве.

## Ключевая формула/инвариант
использование метода .get для безопасной работы с хеш-таблицей.

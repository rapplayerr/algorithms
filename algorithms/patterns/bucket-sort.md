# [bucket-sort]

## Когда применять
Диапазон значений известен и ограничен, нужно сгруппировать по диапазонам, можно распределить по корзинам без сравнения

## Основная идея
Разложить элементы по корзинам (ведрам), отсортировать каждую корзину отдельно, затем объединить.

## Ключевая формула/инвариант
Каждый элемент гарантированно попадает в правильное ведро, и ведра расположены в правильном порядке относительно друг друга.

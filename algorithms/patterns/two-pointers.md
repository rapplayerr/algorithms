# [Два указателя]

## Когда применять
Работа с отсортированными массивами, палиндромы и симметрия, работа с границами (sliding window), сравнение элементов из разных частей, связные списки (быстрый и медленный указатель)

## Основная идея
Вместо того чтобы для каждого элемента запускать второй цикл (O(n²)), мы двигаем два указателя синхронно, обрабатывая каждый элемент не более двух раз (O(n)).
